Cycle It

Privacy Policy 隐私政策

Last Updated: 2025-02-28 最后更新日期：2025-02-28

Cycle It ("the App") is developed and maintained by the individual developer Siriusq. This application is fully open-source, free to use, and designed to help users track recurring item usage cycles. We take your privacy seriously. This Privacy Policy explains how the App handles data.

Cycle It（以下简称“本应用”）由个人开发者 Siriusq 开发与维护。本应用完全开源、免费使用，旨在帮助用户管理物品使用周期。我们非常重视用户隐私，因此特此声明以下隐私政策。

1. What data does the App collect? 1. 本应用收集哪些数据？

The App collects ZERO personal data.

  • No personally identifiable information
  • No analytics
  • No telemetry
  • No data transmission to servers
  • No network communication (except when the user manually exports/imports a local database and upload it)

All item data and usage records are stored locally on your device only.

本应用不收集、存储、传输任何个人信息。

  • 不会收集任何可识别用户身份的数据
  • 不会上传任何信息到服务器
  • 不会使用分析工具或第三方 SDK
  • 不会访问网络（除非用户自行导入/导出本地数据库到云端）

所有数据（物品信息、使用记录、标签等）仅保存在您的设备本地。

2. How is the data used? 2. 本应用如何使用数据？

Local data is used exclusively for:

  • Displaying item information and usage records
  • Predicting the next cycle
  • Scheduling local notifications

No data is ever sent to the developer or any third party.

所有数据均仅用于应用的本地功能，例如：

  • 显示物品及使用记录
  • 预测下次使用周期
  • 安排本地通知提醒

本应用不会将数据发送给开发者或任何第三方。

3. Permissions 3. 权限说明

The App may request the following permissions:

Notification Permission

Used only for local reminders.

File Access Permission (Android only)

Used solely for importing/exporting the local database.

No permission is used for tracking or data collection.

本应用可能会请求以下权限，用于实现功能：

通知权限（Windows / Android / macOS / iOS）

用于发送本地提醒，不涉及网络推送。

文件访问权限（Android）

用于用户手动导入或导出本地数据库文件。

这些权限仅用于对应功能，绝不会用于其他用途。

4. User Data Control 4. 数据的控制权

You have full control over your data:

  • You may delete any item or record at any time
  • Uninstalling the App erases all local data
  • Data is never stored in the cloud unless YOU manually export and upload it

您对自己的数据拥有完全控制权：

  • 可随时删除物品或记录
  • 可随时卸载应用以清除全部数据
  • 数据不会保存在云端（除非用户自行导出到 OneDrive/iCloud 等）

5. Third-Party Services 5. 第三方服务

The App uses no third-party services, such as:

  • Analytics
  • Ads
  • Cloud services

本应用 不使用任何第三方服务。

无分析、无广告、无后端。

6. Policy Updates 6. 隐私政策的更新

Any updates to this policy will be posted on this page. Updates will not change the App’s core principle:

local storage only, offline, no data collection.

如隐私政策更新，将在本页面同步更新。更新不会影响本应用“本地存储、不开网络、不收集数据”的核心原则。

如有任何问题，可通过以下方式联系开发者：
